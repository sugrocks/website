Technical stuff
This website is hosted on a Raspberry-like server at Scaleway, Paris.
The "C1" server is running an ARMv7 CPU with 4 cores and 2GB of RAM.
There's two volumes attached to the server, the first one with the OS and basic stuff (with 50GB) and a second one with 100GB for hosting the files.
The server is Running on Debian and page are served from a Caddy webserver, behind Cloudflare as a CDN.
I generate static files using Python3 scripts and Jinja2 as a template engine.
